2 - Configure SaintCoinach.Cmd.dll.config
      <setting name="DataPath" serializeAs="String">
        <value>D:\SteamLibrary\steamapps\common\FINAL FANTASY XIV Online</value>
      </setting>

4 - Run the following command in the SaintCoinach.Cmd directory to export item data:
